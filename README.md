# 😀 Face Detection and Recognition

## 📌 Description

This project is a real-time **Face Detection and Recognition** application developed using **Python** and **OpenCV**.

It uses the **Haar Cascade Classifier**, a pre-trained face detection model, to detect human faces from a webcam feed and draw bounding boxes around detected faces.

This project was developed as part of the **CodSoft Artificial Intelligence Internship**.

---

## 🚀 Features

- Real-time Face Detection
- Webcam Support
- Detect Multiple Faces
- Fast Face Detection
- Easy to Use
- Beginner Friendly

## 🎥 Output

- Opens the webcam.
- Detects human faces in real time.
- Draws a green rectangle around each detected face.
- Press **Q** to exit the application.
